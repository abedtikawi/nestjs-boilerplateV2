## Installation

```bash
$ npm install
```

## Running the app

```bash
# testing mode
$ source src/secrets/env.testing.sh && npm run start

# development mode
$ source src/secrets/env.development.sh && npm run start:dev

# production mode
$ source src/secrets/env.production.sh && npm run start:prod
```

## Running the app in REPL mode

```bash
# development
$ npm run repl

```
